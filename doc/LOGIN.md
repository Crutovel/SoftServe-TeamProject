# API Documentation for /Login End Point

## Login

**URL :** /login

**Method :** POST

**URL Params :** -

**Data Params :** 

Required: `username=[String], password=[String]`

Example: `username=OlegShvets, password=ghd22df`

**Data type :** form-data

**Success Response:** Code: 200

**Error Response :** Code: 401 Unauthorized
